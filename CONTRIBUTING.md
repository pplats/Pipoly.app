# Contributing to Pipoly.app

We welcome contributions from the community to help improve Pipoly.app. Your insights and efforts are invaluable in making our platform more robust, user-friendly, and comprehensive. This document outlines the guidelines for contributing to this public repository.

## How to Contribute

### Bug Reports

If you find a bug, please report it by opening a new issue on our [GitHub Issues](../../issues) page. Before submitting, please ensure:

*   The bug has not already been reported.
*   You provide a clear and concise description of the problem.
*   You include steps to reproduce the bug reliably.
*   You specify your device, operating system, and browser (if applicable).
*   You attach relevant screenshots or screen recordings.

### Feature Requests

We are always looking for new ideas to enhance Pipoly.app. If you have a feature in mind, please submit it as a feature request on our [GitHub Issues](../../issues) page, or discuss it in our [GitHub Discussions](../../discussions).

When submitting a feature request, please:

*   Clearly describe the proposed feature and its intended benefits.
*   Explain how it would improve the user experience or platform functionality.
*   Provide any mockups or examples if available.

### General Feedback and Questions

For general feedback, questions, or discussions that do not fit into bug reports or feature requests, please use our [GitHub Discussions](../../discussions) forum. This is a great place to connect with other users and the Pipoly.app team.

## Code Contributions

This public repository does not contain the source code for Pipoly.app. Therefore, direct code contributions (e.g., pull requests with code changes) are not applicable here. However, your contributions to documentation, issue triage, and community support are highly valued.

## Code of Conduct

We expect all contributors to adhere to our Code of Conduct. Please treat everyone with respect and create a positive and inclusive environment.

## Thank You

Thank you for your interest in contributing to Pipoly.app. Your efforts help us build a better platform for everyone!
